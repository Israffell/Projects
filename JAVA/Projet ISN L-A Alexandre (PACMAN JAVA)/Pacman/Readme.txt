Pour lancer le progamme il faut : 

	- processing 3.3.7 ou plus
	- mettre la librairie fournie dans \processing-3.3.7\modes\java
	- installer Minim : sur processing allez dans Sketch -> Importer une librairie -> Ajouter une librairie -> rechercher "Minim" (An audio library that provides easy to use classes for playback, recording, analysis, and synthesis of sound).
	- ouvrez le fichier Pacman.pde puis "Exécuter"

Amusez-vous bien !

Ps: Lorsque vous vous voulez activer ou couper la musique, un bouton s'affiche en haut à droite de la fenêtre :)
